## Usage

This shader has several variables that you can use to control the final effect.

| Variable  | Type  | Default | Description |
|-----------|-------|---------|-------------|
| u_color   | int   | 0       | Sepia selection (0 to 1)
| u_opacity | float | 0.75    | Opacity of filter
